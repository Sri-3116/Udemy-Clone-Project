# Udemy-Clone-Project
This project is a Udemy Clone built using HTML5, CSS3, and Flexbox. The website replicates the basic user interface of the Udemy learning platform, providing a clean and responsive design for browsing online courses.

Features
Responsive navigation bar
Course categories section
Recommended courses display
Popular courses section
Search bar design
Attractive course cards with images, ratings, and pricing
Responsive layout using Flexbox
Modern and user-friendly interface
Technologies Used
HTML5
CSS3
Flexbox
Learning Outcomes

Through this project, I improved my understanding of:

Semantic HTML structure
CSS styling and layouts
Flexbox for responsive design
Web page organization and UI development
Building real-world front-end projects
